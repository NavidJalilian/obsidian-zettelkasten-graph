üöÄ How to Use

Getting Started

  1. Install the Zettelkasten Graph plugin in Obsidian
  2. Open the graph view from the command palette
  3. Start dragging nodes to see the magic happen!

Basic Operations

Make a Child Relationship: 1. Click and drag node "3.1" 2. Hover over node "2" 3. See the blue pulsing circle (valid drop zone) 4. Release to make "3.1" become "2.1" Invalid Operations: - Dragging a parent onto its child shows red warning - Self-drops are automatically prevented - Visual feedback guides you to valid operations

Keyboard Shortcuts

Ctrl+Y

Ctrl+Shift+Z
